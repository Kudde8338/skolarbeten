== 21. sum_to_with_step

`sum_to_with_step` Tar två input: ett positivt heltal samt en steglängd och returnerar summan av alla heltal från 0 till och med talet räknat med steglängden.


[source, ruby]
.Exempelanrop
----
sum_to_with_step(10, 2) #=> 30
sum_to_with_step(15, 3) #=> 45
sum_to_with_step(20, 5) #=> 50
sum_to_with_step(7, 1)  #=> 28
sum_to_with_step(5, 2)  #=> 6
----
