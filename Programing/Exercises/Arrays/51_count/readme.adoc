== 51. count

`count` tar en array och ett värde som input och ger antalet förekomster av värdet i arrayen som output.

[WARNING]
====
Skriv funktionen utan att använda metoden `.count`.
====

[source, ruby]
.Exempelanrop
----
count([1, 2, 3, 1, 1], 1) #=> 3
count([1, 2, 3, 1, 1], 2) #=> 1
count([1, 2, 3, 1, 1], 4) #=> 0
count(["a", "b", "a", "a"], "a") #=> 3
count(["a", "b", "a", "a"], "b") #=> 1
----

[IMPORTANT]
====
Använd verktygen link:../goals_and_plans#indexering[indexering] 
====



* Skapa ett flödesschema med penna och papper.
* Testkör flödesschemat med olika värden för att säkerställa att det fungerar.
* Implementera funktionen `count` i `count.rb`.
* Testa din lösning manuellt genom att anropa funktionen med olika argument. 
* Kör de automatiska testerna med `rake` i uppgiftens rotmapp i terminalen.
