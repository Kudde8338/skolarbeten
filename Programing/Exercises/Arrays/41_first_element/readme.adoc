== 41. First Element
`first_element` tar en array som input och returnerar det första elementet i arrayen som output.

[source, ruby]
.Exempelanrop
----
first_element([1, 2, 3]) #=> 1
first_element(["a", "b", "c"]) #=> "a"
first_element([4, 5, 6]) #=> 4
first_element([]) #=> nil
first_element(["a", 1, 2]) #=> "a"
----

[IMPORTANT]
====
Använd verktygen link:../goals_and_plans#indexering[indexering] 
====

[WARNING]
====
Om listan är tom ska funktionen returnera `nil`.
====
