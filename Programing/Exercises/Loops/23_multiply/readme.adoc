== 23. Multiply

`multiply` tar två positiva heltal som input, utan att använda multiplikations-operatorn (`$$*$$`) returnerar produkten av talen.


[source, ruby]
.Exempelanrop
----
multiply(2, 3) #=> 6
multiply(5, 0) #=> 0
multiply(10, 1) #=> 10
multiply(3, 4) #=> 12
multiply(2, 5) #=> 10
----

====
