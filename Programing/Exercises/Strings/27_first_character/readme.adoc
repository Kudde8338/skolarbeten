== 27. first_character

`first_character` tar en sträng som input input och som returnerar det första tecknet i strängen.


[source, ruby]
.Exempelanrop
----
first_character("hello") #=> "h"
first_character("world") #=> "w"
first_character("ruby")  #=> "r"
first_character("code")  #=> "c"
first_character("test")  #=> "t"
----
