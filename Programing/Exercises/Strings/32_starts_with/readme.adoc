== 32. Starts with

`starts_with` tar en sträng och ett tecken (en sträng med enbart ett tecken i) som input och returnerar true om den första strängens första tecken är tecknet i den andra strängen. Annars skall funktionen returnera false.

[source, ruby]
.Exempelanrop
----
starts_with("hello", "h") #=> true
starts_with("ruby", "r") #=> true
starts_with("example", "e") #=> true
starts_with("foo", "b") #=> false
starts_with("concat", "d") #=> false
----

====
