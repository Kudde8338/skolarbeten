== 35. Contains char

`contains_char` tar en sträng och ett tecken (en sträng med enbart ett tecken i) som input och returnerar true om den första strängens innehåller tecknet i den andra strängen. Annars skall funktionen returnera false.

[source, ruby]
.Exempelanrop
----
contains_char("hello", "o") #=> true
contains_char("ruby", "y") #=> true
contains_char("example", "e") #=> true
contains_char("foo", "b") #=> false
contains_char("concat", "d") #=> false
----
