== 37. Chomp

`chomp` tar en sträng som input. Om strängen slutar med en radbrytning (`\n`) eller mellanrum (`" "`) ska funktionen returnera en ny sträng bestående av den ingående strängen, men utan radbrytningen eller mellanrummet. Om strängen inte slutar med en radbrytning returneras den ingående strängen oförändrad.

[source, ruby]
.Exempelanrop
----
chomp("hello") #=> "hello"
chomp("hello\n") #=> "hello"
chomp("hell\no") #=> "hell\no"
chomp("hell\no\n") #=> "hell\no"
----
