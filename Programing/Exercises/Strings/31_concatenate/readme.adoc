== 31. Concatenate

`concatenate` tar två strängar som input och returnerar en ny sträng som är de två strängarna konkatenerade.

[source, ruby]
.Exempelanrop
----
concatenate("hello", "world") #=> "helloworld"
concatenate("ruby", "programming") #=> "rubyprogramming"
concatenate("example", "test") #=> "exampletest"
concatenate("foo", "bar") #=> "foobar"
concatenate("concat", "enate") #=> "concatenate"
----
