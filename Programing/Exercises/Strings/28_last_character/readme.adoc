== 28. Last character

`last_character` tar en sträng som input input och returnerar det sista tecknet i strängen.

[source, ruby]
.Exempelanrop
----
last_character("hello") #=> "o"
last_character("world") #=> "d"
last_character("ruby") #=> "y"
last_character("programming") #=> "g"
last_character("a") #=> "a"
----
